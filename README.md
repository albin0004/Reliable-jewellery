
# Reliable Jewellery Quotation App

This is a static web application. To share it publicly, you need to deploy it to a web host.

## Quickest Deployment (Surge.sh)

1. Open your terminal in this folder.
2. Run the following command:
   ```bash
   npx surge
   ```
3. Follow the prompts (enter email/password to create a free account).
4. It will give you a public URL (e.g., `flawless-jewel.surge.sh`).

## GitHub Pages

1. Initialize a git repository: `git init`
2. Commit files: `git add . && git commit -m "Initial commit"`
3. Create a repo on GitHub and push.
4. Go to Repo Settings -> Pages -> Select 'main' branch.

## Netlify / Vercel

- **Netlify Drop**: Drag this folder to [app.netlify.com/drop](https://app.netlify.com/drop).
- **Vercel**: Run `npx vercel` in the terminal.
