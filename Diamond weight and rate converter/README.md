# Diamond Rate & Weight Converter

A professional quotation web application for diamond jewelry, featuring real-time rate conversion and dynamic weight/price calculations.

## Features

- **Rate Convention Header**: Calculates 18K Gold Price based on USD, Purity, and Dirham exchange rate.
- **Dynamic Table**: 
  - Manage 21+ rows of diamond inputs.
  - Automatic grouping and color-coding by USD Rate.
  - Real-time calculations for Weight and Total Price in AED.
- **Visuals**: Premium glassmorphism design with responsive layout.
- **Export/History**: (To be added: currently supports real-time editing and adding new rows).

## How to Use

1. **Open the Application**:
   Simply open `index.html` in any modern web browser (Chrome, Edge, Firefox).

2. **Rate Convention**:
   - Enter **USD Gold Amount**, **Purity**, and **Dirham Value** in the top panel.
   - The **18K Gold Price** updates automatically.

3. **Diamond Table**:
   - The table is pre-loaded with standard sizes.
   - Enter the **PCS (Quantity)** for any row to calculate totals.
   - Change the **USD Rate** to re-sort and re-group rows automatically.
   - Click **Add New Entry** to add custom rows at the bottom.

## Technology
